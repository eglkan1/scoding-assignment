== Description ==

**WPForms is a great form builder plugin but it misses some of the features**. Booster for WPForms brings those missing features and enhances your experiecnce with WPForms.

Here is the list of features that are added by Booster for WPForms
* Ability to mark fields as readonly
* Prevent browser autocomplete for forms
* Show remaining entries at top of form
* Option to set Max characters for input and textarea fields
* Option to set values for checkbox, radio and dropdown fields
* Min and Max range for numbers field
* Ability to change validation message per field

= Premium Addons to Extend the Functionality =
* [ **Range Slider** ](https://wpmonks.com/downloads/range-slider-for-wpforms/)
**Adds a range slider field in WPForms editor**. You can set custom steps, fixed start values , add prefix and suffix for values, fill color, enable grid/pips/scale, enable second handle set start value for second handle and a lot more.

* [ **Popup for WPForms** ](https://wpmonks.com/downloads/popup-for-wpforms/)
**With this addon you can show WPForms in Popup/modal**. You can create unlimited popups for different forms or with different settings for same form. The popup can be opened automatically or on click of button. The design and layout of popup can be easily customized without any coding.

* [ **Submission Preview** ](https://wpmonks.com/downloads/confirmation-step-for-wpforms/)
**With this addon you can show submission preview of all data filled by users**. It can act as a confirmation step so, that they can check everything they have entered at one place. They can also go back and fix if any field is not filled correctly. This will prevent mistakes in form submission.

* [ **Address Autocomplete for WPForms** ](https://wpmonks.com/downloads/address-autocomplete-for-wpforms/)
**With this addon you can enable address autocomplete feature for single line text and address fields**. User will start getting relevant address suggestions as they type in the fields.

== Installation ==


1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= How do I enable Boosts =

You can enable boosts by navigating to Forms -> Settings -> Booster.
